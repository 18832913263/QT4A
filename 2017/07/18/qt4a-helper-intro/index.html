QT4A助手功能简介

发表于 2017-07-18
  1. 解锁屏幕
  2. 模拟相机，用以解决不同系统中相机不统一导致无法自动化问题
  3. 浏览器，可用于进行终端页面的测试
  4. 系统测试桩服务（用于非Root手机）
  5. 辅助点击功能，可以点掉授权弹窗等意外系统弹窗
  6. 输入法，用于发送按键
